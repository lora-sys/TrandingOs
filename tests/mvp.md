## goal 
make a useful mvp for the Trading Pi agent
Trading Pi 的目标，不是替你交易，而是帮助一个普通人，从“不会交易”，成长为“拥有自己 Alpha 的交易者”。

single agent + workflow+ skills + artifact management

## sucess criteria
- User can input a trading query and receive a comprehensive trade plan
- The agent can successfully execute the trade plan with the help of the workflow engine and skill registry
- All artifacts are generated and displayed in the UI for user review
- The user can approve or reject each step of the execution timeline before proceeding
